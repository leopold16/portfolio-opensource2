Espresso Teaching Website

Group project for UI Design class, focused on teaching users how to make different types of espresso.

Project Overview

This project was developed as part of a UI Design class. Working in a group, we created an interactive website aimed at coffee enthusiasts who want to learn how to make various types of espresso, including lattes, cappuccinos, macchiatos, and more. The goal was to create an engaging and visually appealing experience for users, allowing them to easily follow along with instructions.

Design and Illustrations

Our team designed custom illustrations to visually represent each coffee type and its preparation steps. These illustrations played a crucial role in making the website more engaging and user-friendly. Each coffee drink has a unique illustration to guide users through the steps required to make it.

Project Video

We also created a demonstration video that showcases the website’s features and user flow. This video explains the website’s design and functionality, and provides an overview of the different types of espresso users can learn to make.

Key Features

  • Interactive tutorials for each type of espresso.
  • Custom illustrations for each coffee type to enhance user engagement.
  • Step-by-step instructions that make it easy for beginners to follow along.

Learning Outcome

This project taught our group valuable lessons in UI design principles, teamwork, and the importance of visual elements in enhancing user experience. By combining detailed instructions with engaging illustrations, we created a website that is both educational and enjoyable for coffee enthusiasts.
